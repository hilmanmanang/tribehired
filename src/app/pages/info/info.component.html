Apps Info

This apps is like Twitter clone, where user can see all posts at home page. Then when user click individual post, user can see all comments from post. In additional, user can filter the comments based on name, email and comment's body.

Technology used: This Apps is made using Angular version 12.0.4, with SCSS preprocessor for styling. The open source code is push to GitHub at https://github.com/hilmanmanang/tribehired.
